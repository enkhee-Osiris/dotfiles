## Installation

Prerequisites:

1. [Xcode Command Line Tools](http://railsapps.github.io/xcode-command-line-tools.html).

Then run these commands in the terminal:

```
git clone git@github.com:enkhee-osiris/dotfiles.git ~/dotfiles
cd ~/dotfiles
./setup/stuff.sh
./setup/zsh.sh
```
